Projects and Coursework

SpaceVentureZ


SpaceVentureZ is a “shoot-em-up” style arcade game that will take players on a laser blastin' journey through our solar system to defeat vicious alien invaders. Featuring a classic top-down view as well as a robust weapon upgrade system, players will experience a retro, two-dimensional simulation with a modern twist!

This project was created for my Software Engineering class. Working in a group with 4 other peers over the course of an entire semester, we had the responsibility of designing and delivering a product which was broken into 5 iterations. Each iteration included planning, work distribution, design, development, and testing.


Post Covid System


This project is a system that is meant to be a solution to the issues that Covid-19 brought to restaurants. It provides restaurants with a way to set up waitlists, take orders from customers, and also provides a way for customers to leave reviews about the restaurant and their services.

This project is web-based and was built with node.js.
